# 🚆 Railway Management System (Python + SQL + Tkinter)

This project is a complete Railway Management System developed using Python (Tkinter) and SQL to efficiently manage train operations and passenger data.

## 🔹 Overview
The system simulates real-world railway operations including ticket booking, cancellation, train management, and passenger record handling with real-time database integration.

## 🔹 Key Features
- 🚆 Add and manage train details  
- 🎟️ Ticket booking and cancellation system  
- 📊 Real-time data storage using SQL database  
- 📁 Passenger record management  
- 🔍 Track total tickets booked  
- 🔐 QR Code-based ticket generation  
- 🖥️ User-friendly GUI built with Tkinter  

## 🔹 Technologies Used
- Python (Tkinter)  
- SQL (Database Management)  

## 🔹 How It Works
- User interacts with GUI to perform operations  
- All actions (booking, cancellation, updates) are stored in SQL database  
- Data is retrieved in real-time for display and reporting  
- QR code is generated for ticket verification  

## 🔹 Project Structure
- Python files (.py) for GUI and logic  
- SQL database for storing data  
- Screenshots for output visualization
